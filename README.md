# Auto Screenshot Tab

Automatically takes screenshots of a selected tab to a local directory.

## Installation Instructions (Google Chrome / Microsoft Edge)

Follow these steps to install the extension manually as an "unpacked" extension:

1. **Download the Extension:** Ensure you have the extension folder on your computer. If it's a `.zip` file, extract it to a known location.
2. **Open Extensions Page:** 
   - In Google Chrome, open a new tab and go to `chrome://extensions/`
   - In Microsoft Edge, open a new tab and go to `edge://extensions/`
3. **Enable Developer Mode:** In the top right corner of the Extensions page, toggle the **Developer mode** switch to the ON position.
4. **Load the Extension:** Click the **Load unpacked** button that appears in the top left corner.
5. **Select the Folder:** A file browser will open. Navigate to the folder containing the extension files (specifically, the folder that contains the `manifest.json` file) and select it.
6. **Pin the Extension (Optional):** The extension is now installed! You can click the puzzle piece icon (Extensions menu) in your browser toolbar and click the pin icon next to "Auto Screenshot Tab" to keep it visible.

## Usage
Once installed, click the extension icon to access its controls and start taking automatic screenshots of your selected tabs.
